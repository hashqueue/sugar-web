# sugar-web

一站式项目研发管理平台 - developed with Vue3、Ant Design Vue 3 and Vite4.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
